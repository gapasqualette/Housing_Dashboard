# 🏠 Housing Dashboard

An interactive data visualization project focused on analyzing housing data, built using **Python and Streamlit** to transform raw real estate data into actionable insights.

---

## 📊 Project Overview

This project provides an interactive dashboard for exploring housing market data, enabling users to analyze pricing patterns, geographic distribution, and key real estate indicators.

The main goal is to demonstrate how data can be transformed into **clear, decision-oriented insights** through visualization and filtering.

---

## 📂 Dataset

The dataset used in this project can be found in this repository:

[https://www.kaggle.com/datasets/kanths028/usa-housing/data]

---

## 🧠 Business Questions

* What factors most influence housing prices?
* How do prices vary by location and property characteristics?
* What is the distribution of properties by price range?
* Are there patterns or outliers in the housing market?

---

## 🛠️ Technologies Used

* **Python (Pandas, NumPy)** → Data cleaning and transformation
* **Matplotlib / Seaborn** → Data visualization
* **Streamlit** → Interactive dashboard development
* **SQLite / CSV** → Data storage and manipulation

## 🔄 Data Pipeline

### 1. 📥 Data Preparation

* Data loading and validation
* Handling missing values
* Data cleaning and formatting

### 2. 🔍 Data Analysis

* Feature exploration (price, location, category)
* Creation of derived variables
* Aggregations and grouping

### 3. 📊 Visualization (Streamlit)

* Interactive charts and KPIs
* Dynamic filters (location, category, date, etc.)
* User-friendly dashboard interface

---

## 📈 Key Metrics (KPIs)

* 💰 Average property price
* 📍 Price distribution by location
* 🏠 Number of properties listed
* 📊 Price range segmentation
* 📉 Trend analysis over time

## 🚀 How to Run the Project

### 1. Clone the repository

```bash id="house2"

```

### 2. Install dependencies

```bash id="house3"

```

### 3. Run the application

